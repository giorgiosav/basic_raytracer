# Instructions

1. Place this code in `simtrax/samples/src/basic_raytracer`
2. To compile, use `make` inside the directory from the terminal. This should create the following files:
 * `objs` directory.
 * `rt-llvm.s`
 * `rt.s`
 * `run_rt`
 * `run_rt.dSYM`
 * `rt.bc`

3. To run, type `./run_rt` in the terminal.
